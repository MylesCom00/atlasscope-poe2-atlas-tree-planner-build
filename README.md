# 📖 About

AtlasScope is an open-source desktop application for **Path of Exile 2** players.

Build Atlas strategies, experiment with different farming setups, organize atlas trees and keep all of your favorite endgame plans in one place.

Designed with performance in mind using **Rust**, **Tauri**, **React** and **TypeScript**.
